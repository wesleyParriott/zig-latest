# ZIG LATEST
_a shell script that updates your zig to the master version_

# How to run
``` 
git clone https://github.com/wesleyParriott/zig-latest.git && \
cd zig-latest && \
./zig-latest
```

# Uses
* jq for json parsing
* curl for get requests
* wget for file download
* sudo for permission escalation
